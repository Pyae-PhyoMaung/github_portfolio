Automated Kidney Stone Detection System
Problem Statement:

Manual identification of kidney stones from CT scans is time-consuming for radiologists and prone to human oversight, creating a critical need for rapid, automated diagnostic support.

Outcome:

Successfully trained and deployed a YOLOv8 object detection model capable of precisely isolating kidney stones in medical imaging. The pipeline included annotating raw CT scans via CVAT into COCO format and applying advanced data augmentation to ensure robust detection accuracy.

Burmese News Headline Generator
Problem Statement:

Generating concise, contextually accurate headlines for Burmese news text is a resource-intensive task. Furthermore, implementing abstractive summarization for low-resource languages like Burmese presents unique challenges due to limited public datasets and complex script tokenization.

Outcome:

Delivered an end-to-end NLP pipeline by fine-tuning a multilingual T5 (mT5) transformer model on a preprocessed Burmese news dataset. Built a live, real-time inference web application using Gradio and validated headline quality using industry-standard ROUGE and BLEU scores.
